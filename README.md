# CS330 ReadME
Before working on this project, I had no experience with 3D modeling or graphics. This project has allowed me to develop new design skills such as breaking down complex objects into simple shapes, applying textures, and lighting. The design process for this project was iterative as I continued to learn from the resources and apply the material to expand the project. As I worked through each milestone, I had to keep testing the piece I was applying to ensure it looked correct and to produce a polished product. This tactic can be applied in future projects as it lets me break down large problems into smaller steps and make continual progress towards a goal.
Computational graphics and visualization give me skills I can use to continue either an educational or a professional path. As computer science is all about learning and continued learning, with almost limitless pathways in the new digital world, having the ability to break down a 3D scene into objects and create a unique 3D scene opens up the more graphics-focused industries like video games, graphic design, or website development. This project also improved my approach to program development through constant iteration and a different set of problem-solving techniques. Through my iterations, I had to test the many different pieces, object scales, and transformations, lighting, texture acquisition, and scaling to reduce tearing.
